    # Scala包：

    包的作用：1、区分相同名字的类 2、当类很多时，可以很好的管理类 3、控制访问范围

    包的命名规范：com.公司名.项目名.业务模块名

    包的使用：

    1、Scala中基本的package包语法和Java完全一致，还有一种是通过嵌套的风格表示层级关系

    package com {
 import com.company.Inner //父包访问子包需要导包
      object Outer {
      val out: String = "out"
      def main(args: Array[String]): Unit = {
      println(Inner.in)
      }
      }
 package company {
      object Inner {
      val in: String = "in"
      def main(args: Array[String]): Unit = {
      println(Outer.out) //子包访问父包无需导包
      }
      }
      }
}

    2、用package关键字可以嵌套声明使用，子包中的类可以直接访问父包中的内容，而无需导包

    3、Scala中package也可以看作对象，并声明属性和函数， 作为其对应包下所有 class 和 object 的共享变量，可以被直接访问，调用方式直接为变量名或者方法名。

    package ccc{
  package ddd{
    object printWay{
      def main(args: Array[String]): Unit = {
        println(school)
        goSchoolStyle("zhangsan")
      }
    }
  }
}

// 定义一个包对象
package object ccc{
  val school: String = "清华大学"
  def goSchoolStyle(name:String)={
    println(s"${name}去${school}上学！")
  }
}

    导包操作：

    1、和Java一样，可以在顶部使用 import 导入，在这个文件中的所有类都可以使用。

    2、局部导入：什么时候使用，什么时候导入。在其作用范围内都可以使用

    3、通配符导入：import java.util._

    4、给类起名：import java.util.{ArrayList=>JL}

    5、导入相同包的多个类：import java.util.{HashSet, ArrayList}

    6、屏蔽类：import java.util.{ArrayList =>,} //引入 java.util 包下屏蔽 ArrayList 类

    7、导入包的绝对路径：new_root_.java.util.HashMap

    package java {
 package util {
 class HashMap {
 }
 }
}

    # Scala类：

    一个 Scala 源文件可以包含多个类、属性、方法、对象、构造器

    # 类中-属性:

    1、下划线表示类的默认初始化

    如：在类中,var age : Int = _ // 下划线表示类的默认初始化(val 修饰的属性不能赋默认值，必须显示指定)

    2、可以通过@BeanProperty给属性赋值getter、setter方法

    //Bean 属性（@BeanProperty）
 @BeanProperty var sex: String = "男"
    1
    2

    3、访问权限

    private : 私有访问权限，只在类的内部和伴生对象中可用。

    private[包名]: 包访问权限，包名下的其他类也可以使用。

    Protected : 受保护权限，同类、子类可以访问，同包无法访问。

    # 类中-方法:

    对于方法的调用，Scala中的类的方法其实就是函数，所以声明方式完全一样，但是使用方式必须使用对象

    # 类中-对象:

    1、创建对象

    基本语法：val | var 对象名 [：类型] = new 类型()

    注意事项：

    （1）val 修饰对象，不能改变对象的引用（即：内存地址），可以改变对象属性的值。

    class Person {
 var name: String = "zhangsan"
}
object Person {
 def main(args: Array[String]): Unit = {
 val person = new Person()
 person.name = "lisi"
 // person = new Person()// 错误的
 println(person.name)
 }
}

    （2）var 修饰对象，可以修改对象的引用和修改对象的属性值

    （3）自动推导变量类型不能多态，所以多态需要显示声明

    # 类中-构造器:

    1、构造方法(构造器)声明

    基本语法

     class 类名(形参列表) { // 主构造器
 // 类体
 def this(形参列表) { }// 辅助构造器
 def this(形参列表) { }//辅助构造器可以有多个...
 }

    2、构造方法(构造器)分类

    1、主构造函数： 如class employee(){}

    2、辅构造函数: 在主构造类中，使用 def this(username:String,age:Int){}

    构造器补充说明

    (1)、主构造函数和多个辅助构造函数，如果主构造器无参数，小括号可省略

    //class Person (){}
class Person {}
    1
    2

    (2)、辅助构造方法不能直接构建对象，必须直接或者间接调用主构造方法

    class employee{//主构造器
  def this(age:Int){
    this()//必须直接或者间接调用主构造方法,假如主构造器有参数，则需要加上参数
    this.age=age
    println("辅助构造器")
    }
}

    3、构造方法(构造器)形参类型

    （1）未用任何修饰符修饰，这个参数就是一个局部变量(未用任何修饰)

    （2）var 修饰参数，作为类的成员属性使用，可以修改(var 修饰)

    （3）val 修饰参数，作为类只读属性使用，不能修改(val 修饰)

    class Employee(name: String, var age: Int, val sex: String) {
}
object Test {
 def main(args: Array[String]): Unit = {
 var employee = new Employee("zhangsan", 18, "男")
 // （1）未用任何修饰符修饰，这个参数就是一个局部变量
 // printf(person.name)
 // （2）var 修饰参数，作为类的成员属性使用，可以修改
 employee.age = 19
 println(employee.age)
 // （3）val 修饰参数，作为类的只读属性使用，不能修改
 // employee.sex = "女"
 println(employee.sex)
 }
}
